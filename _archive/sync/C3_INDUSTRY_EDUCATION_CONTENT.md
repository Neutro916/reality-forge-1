## EDUCATION-SPECIFIC MANIPULATION PATTERNS

### Pattern 1: Mission Shield
"We're doing this for the students."
Using student welfare to shut down legitimate concerns.

**Recognition:** Mission invoked to bypass discussion of working conditions, resources, or decisions.
**Transformation:** Student welfare AND staff welfare. Both matter.

### Pattern 2: Martyrdom Expectation
"Teachers don't do it for the money."
Using calling/vocation to justify exploitation.

**Recognition:** Passion weaponized against fair compensation and boundaries.
**Transformation:** Dedication deserves support, not exploitation.

### Pattern 3: Expertise Dismissal
"You don't have an EdD, so..."
Using credentials to silence practitioners.

**Recognition:** Academic credentials trump classroom experience.
**Transformation:** Multiple forms of expertise valued.

### Pattern 4: Parent Pressure Transfer
"Parents are demanding this."
Using parent concerns to avoid accountability.

**Recognition:** Passing pressure down instead of managing up.
**Transformation:** Leaders buffer, not transfer pressure.

### Pattern 5: Data Manipulation
"The data shows..." (selectively)
Using assessment data to support predetermined conclusions.

**Recognition:** Data cherry-picked to justify decisions.
**Transformation:** Complete data, honest interpretation.

### Pattern 6: Compliance as Control
"It's required by [authority]."
Using regulations to avoid explaining decisions.

**Recognition:** Policy as shield rather than genuine constraint.
**Transformation:** Explain reasoning, not just rules.
