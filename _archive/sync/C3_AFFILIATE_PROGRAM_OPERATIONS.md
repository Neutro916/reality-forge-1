## OPERATIONAL SYSTEMS

### Tracking Technology:

**Recommended Platforms:**
- Rewardful (Stripe integration)
- FirstPromoter
- PartnerStack (enterprise)
- Custom build (long-term)

**Required Features:**
- Unique link generation
- Cookie tracking (60 days)
- Commission calculation
- Payout management
- Fraud detection
- Reporting dashboard

### Attribution Rules:

1. **First touch wins** - First affiliate to refer gets credit
2. **30-day cookie for referrals, 60-day for affiliates**
3. **Direct beats affiliate** - If customer returns directly within 7 days, affiliate still credited
4. **No stacking** - Can't combine affiliate + referral credit
5. **Disputes resolved by Character Forge** - Final decision

### Fraud Prevention:

**Automated Detection:**
- Multiple purchases from same IP
- Rapid cookie changes
- Self-referral attempts
- Unusual conversion patterns
- Chargebacks/refunds >10%

**Manual Review Triggers:**
- First payout >$200
- Sudden traffic spikes
- Conversion rates >20%
- Geographic anomalies
- New affiliate >$500 in first month

**Consequences:**
- First offense: Warning + audit
- Second offense: Commission hold + review
- Third offense: Program termination
- Fraud confirmed: Forfeiture + potential legal
