## BATTLECARD 1: VS. FRANKLIN COVEY / DALE CARNEGIE

### What They Do Well:
- Strong brand recognition
- Established corporate presence
- Proven content (7 Habits, etc.)
- Large facilitator network
- Comprehensive curriculum

### Where We Differentiate:

| Factor | Traditional Training | Character Forge |
|--------|---------------------|-----------------|
| Focus | Skills and habits | Pattern transformation |
| Approach | Teach techniques | Transform root causes |
| Duration | Event-based | Sustained practice |
| Personalization | Generic content | Individual assessment |
| Measurement | Satisfaction surveys | Behavioral change |
| AI Enhancement | None | AI Coach integration |

### Key Messaging:

**When prospect mentions them:**
"Franklin Covey and Dale Carnegie do excellent work on leadership skills. The question is: what's underneath the skills? If someone has patterns that undermine trust—urgency manipulation, guilt-tripping, authority plays—adding skills on top doesn't solve the root issue. We work at the pattern level, which makes skill application actually stick."

### Objection Handling:

**"We already do Franklin Covey training."**
"That's great foundation. How are those skills landing? Are leaders actually applying them consistently? Often, underlying patterns interfere with skill application. Character Forge addresses those patterns, which amplifies everything else you're doing."

**"They're the industry standard."**
"They are, and for good reason. They've helped millions of leaders. We serve a different purpose—not teaching skills, but transforming the patterns that prevent skills from being applied. Many clients do both."
