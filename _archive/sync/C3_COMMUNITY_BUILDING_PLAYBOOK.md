## MODERATION GUIDELINES

### Community Standards:

1. **Respect** - Treat all members with dignity
2. **Honesty** - Speak truth, even when uncomfortable
3. **Confidentiality** - What's shared here stays here
4. **Support** - We're here to help, not judge
5. **Growth** - We're all works in progress
6. **Non-manipulation** - Practice what we learn

### Moderation Actions:

| Issue | Response |
|-------|----------|
| Off-topic post | Move to appropriate space |
| Mild violation | Private message reminder |
| Repeated violation | Warning with consequences |
| Serious violation | Temporary suspension |
| Egregious violation | Permanent removal |

### What Gets Removed:

- Spam/promotion without permission
- Personal attacks
- Confidentiality breaches
- Manipulation tactics being advocated
- Illegal content
- Repeated off-topic posting

### Conflict Resolution:

1. **Private first** - Address issues in DM
2. **Assume good intent** - Most conflicts are misunderstandings
3. **Facilitate dialogue** - Help parties understand each other
4. **Clear decisions** - When needed, decide and communicate
5. **Learn and improve** - Update guidelines if needed

- Reinforce community standards
- Private conversations with violators
- Public reminder of values
