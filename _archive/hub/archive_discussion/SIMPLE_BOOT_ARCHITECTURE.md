## TRINITY COMMUNICATION PROTOCOL (Simple Version)

### Boot Sequence
```
1. Instance wakes up
2. Reads START_HERE.md (30 seconds)
3. Checks hub/INBOX.md for messages
4. Reads hub/CURRENT_TOPIC.md for context
5. Adds thoughts to hub/DISCUSSION.md
6. Waits for responses before acting
```

### Message Format
```
## [INSTANCE] - [TIMESTAMP]

**SITUATION:** What I see
**THINKING:** What I think about it
**QUESTION:** What I need from others
**PROPOSAL:** What I suggest (optional)
```

### Response Protocol
```
- READ before responding
- THINK before building
- ASK if unclear
- CHALLENGE if you disagree
- BUILD only after consensus
```
